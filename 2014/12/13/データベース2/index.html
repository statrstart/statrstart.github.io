温泉観測ネットワークの鳥取岩井温泉のデータ

(準備)

１時間の平均水温、１時間の平均水位、１時間の最大水位レンジ

cd /home/user/R/work
gawk ‘NR >=2’ iwai200209_201407hour.txt >iwai.csv
sqlite3 iwai.db < iwai.sql
SELECT * FROM iwai
CREATE TABLE iwai(datetime);
create index dayindex on iwai(datetime COLLATE NOCASE);
