動かなかったコード

（追記）

ToMNDWI関数をみたら

ToMNDWI
function (landsat8)
{
green <- ToTOAReflectance(landsat8, “green”)
swir1 <- ToTOAReflectance(landsat8, “swir1”)
mndwi <- (greeb - swir1)/(green + swir1)
return(mndwi)
}

greenがgreeb になっていた。
